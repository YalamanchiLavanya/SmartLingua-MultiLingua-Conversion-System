# 🌍 SmartLingua – Multilingual Conversion System
# 📖 Overview
SmartLingua is a Python-based multilingual conversion application built with Streamlit. It provides an all-in-one solution for translating text, documents, and speech into multiple languages while also converting translated text into speech. The application supports PDF, DOCX, microphone input, and audio file processing, making language translation fast, accurate, and accessible.

# ✨ Features
🔐 Secure user authentication with username and password
🔑 Credential recovery using security questions
🌐 Supports translation into 45+ languages
📝 Text-to-Text Translation
🔊 Text-to-Speech Conversion
📄 PDF/DOCX Document-to-Text Translation
📑 PDF/DOCX Document-to-Speech Conversion
🎤 Live Speech-to-Text Translation using Microphone
🔈 Live Speech-to-Speech Translation
🎵 Audio File-to-Text Translation (WAV/MP3)
🎧 Audio File-to-Speech Translation
⬇️ Download translated text files
⬇️ Download translated speech (MP3)
🎨 Attractive Streamlit interface with customized backgrounds
